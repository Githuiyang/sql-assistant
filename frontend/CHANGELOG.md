# 更新日志

所有重要变更都会记录在此文件中。

## [0.1.0] - 2025-01-03

### 🎉 首次发布

#### 新增功能
- **数据导入模块**
  - 支持粘贴SQL CREATE TABLE语句
  - 支持上传CSV文件（拖拽上传支持）
  - 实时校验SQL语法和文件格式
  - 自动验证文件数量和大小（单个≤10MB）

- **字段字典生成**
  - 基于SQL和CSV自动生成字段字典
  - 智能识别表名、字段、数据类型
  - 自动推断表关联关系
  - 支持在线编辑字典内容
  - 支持添加/删除表和字段
  - 导出字典为CSV格式

- **SQL智能生成**
  - 自然语言输入生成SQL
  - 严格基于字段字典生成
  - SQL语法校验（sql.js）
  - 代码高亮显示
  - 复制和下载SQL功能
  - 错误反馈和自动修复

- **查询建议功能**
  - 分析字段字典生成10个查询场景建议
  - 分类展示（统计分析、趋势分析、用户画像等）
  - 显示业务价值说明
  - 一键使用建议跳转生成

- **历史记录管理**
  - 保存所有导入记录
  - 保存SQL生成历史
  - 按时间筛选和删除
  - 7天自动清理机制

- **大模型支持**
  - OpenAI (GPT-4/GPT-3.5)
  - Anthropic (Claude)
  - 阿里云千问
  - Moonshot Kimi
  - 智谱AI
  - DeepSeek

- **用户体验**
  - 首次使用引导（3步骤）
  - 帮助文档组件（各页面独立帮助）
  - 友好的错误提示系统
  - 示例数据快速体验
  - 快捷键支持（Ctrl+Enter生成等）

#### 技术特性
- **前端框架**: Next.js 14 (App Router)
- **UI组件**: shadcn/ui + Radix UI
- **样式方案**: TailwindCSS
- **数据库**: IndexedDB (本地存储)
- **SQL校验**: sql.js (SQLite WebAssembly)
- **设计风格**: 清冷科技风（深色主题）
- **动画效果**: Framer Motion

#### 部署支持
- Docker容器化部署
- docker-compose一键启动
- 生产环境优化构建
- 环境变量配置模板

#### 文档
- 完整的README.md（中文+英文）
- MIT开源协议
- 贡献指南（CONTRIBUTING.md）
- 更新日志（CHANGELOG.md）

#### 已知限制
- 仅支持 SELECT 查询生成
- 需要手动配置大模型API密钥
- CSV文件大小限制≤10MB
- 数据7天后自动清理
